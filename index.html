前回の振り返り

AI コンサルティング キックオフセッション ・ 2026-06-16 ・ Teams

テーマは「今やっていること / 達成したいこと」と「AI エージェントの基礎的な考え方」のすり合わせ。土台となる思考が無いとエージェントは機能しない、という前提でコア概念と実践の両方をカバーしました。

決定事項

  • 本セッションは目標とエージェント基礎のキックオフと位置づける
  • Claude Code を採用し、CLI セットアップから始める
  • アドホックな Q&A をやめ、まず CLAUDE.md を作成・改善する
  • markdown の管理・閲覧に Obsidian を使う
  • 反復ワークフローは再利用可能なスキルにする
  • 来週フォローアップを実施（日程はクライアント側で予約）

押さえた考え方

  • Q&Aで終わらせない：毎回トリガーしなくても物事が進む状態を作る（自動化・継続・記憶・自走）
  • CLAUDE.md = システムプロンプト：AI の役割・人格の中核。長すぎると効かない（100〜200行）
  • スキル化：椅子が欲しいなら毎回木から始めず「椅子スキル」を用意する
  • メタ思考が役割：どう指示し、記憶させ、足場を組むかを考え続ける
  • ツール非依存：文脈をローカルに持てば、どのAIでも持ち運べる
